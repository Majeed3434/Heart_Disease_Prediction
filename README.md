# Heart Disease Prediction

## Project Overview
This project predicts whether a person has heart disease based on health parameters using machine learning models.

## Dataset
- Features: Age, Sex, Blood Pressure, Cholesterol, Max Heart Rate, etc.
- Target: Presence of Heart Disease (0 = No, 1 = Yes)

## Models Used
- Logistic Regression
- Decision Tree

## How to Run
1. Install dependencies:
   
   pip install -r requirements.txt
